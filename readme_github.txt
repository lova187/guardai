# 🛡️ GuardAI - אימון הגנה עצמית בעזרת בינה מלאכותית

> **GuardAI** הוא אפליקציה מתקדמת לאימון הגנה עצמית המשתמשת בבינה מלאכותית לניתוח תנועות בזמן אמת ומתן משוב מיידי לשיפור הטכניקה.

## 🌟 תכונות עיקריות

### 🎯 **אימון חכם עם AI**
- ניתוח תנועות בזמן אמת דרך המצלמה
- משוב מיידי על טכניקת העמידה, בלוקים ואגרופים
- תיקון טעויות באופן אוטומטי ("תרים הגנה", "תוריד סנטר")

### 🔥 **אימון תחת לחץ**
- סימולציות לחץ עם צלילים והוראות מפתיעות
- תרגול להתמודדות עם מצבי "קיפאון"
- הכנה פסיכולוגית למצבים אמיתיים

### 📊 **מעקב התקדמות מתקדם**
- סטטיסטיקות מפורטות לכל סוג תרגיל
- גרפים ויזואליים של שיפור לאורך זמן
- מערכת הישגים ואתגרים

### 🏆 **גיימיפיקציה**
- תגים והישגים מותאמים אישית
- אתגרים שבועיים וחודשיים
- מערכת נקודות ורמות

## 🚀 התחלה מהירה

### דרישות מערכת
- דפדפן מודרני עם תמיכה ב-WebRTC
- מצלמה פועלת
- חיבור אינטרנט יציב (לטעינת מודלי AI)

### התקנה

```bash
# שיבוט הריפו
git clone https://github.com/yourusername/guardai-self-defense.git
cd guardai-self-defense

# התקנת dependencies
npm install

# הרצה במצב פיתוח
npm run dev

# בנייה לפרודקשן
npm run build
```

### שימוש מהיר

1. **התחלה** - פתח את האפליקציה ועבור את שאלון האונבורדינג
2. **אימון** - אפשר גישה למצלמה והתחל באימון הראשון
3. **התקדמות** - עקוב אחר השיפור בלשונית "התקדמות"
4. **אתגרים** - השלם אתגרים שבועיים לתגים נוספים

## 🛠️ טכנולוגיות

### Frontend
- **React 18** - UI framework מודרני
- **Tailwind CSS** - עיצוב רספונסיבי
- **Lucide React** - אייקונים מעוצבים
- **Vite** - בילד מהיר ואופטימלי

### AI & Computer Vision
- **MediaPipe** - ניתוח תנועות בזמן אמת
- **Pose Detection** - זיהוי תנוחות גוף
- **WebAssembly** - ביצועים מקסימליים
